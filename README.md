### General
* Filesystem
* WikiGraph
* IDE
* MCP Host

## Setup

1. Install Python dependencies:

```bash
pip install -r requirements.txt
```

2. Copy the configuration file:

```bash
cp config.yaml.example config.yaml
```

3. Update `config.yaml` with your settings.

4. Run the application:

```bash
python app.py
```
